# Project Instructions

## Goal
Build a fast multi-page portfolio website that showcases Jaden's design and UI/UX skills.

## Product Direction
- This is a portfolio website, not a generic template site.
- The site should communicate strong taste in design, layout, typography, and interaction design.
- Every major page and component should reflect polished UI/UX thinking.

## Priorities
- Keep the website snappy.
- Keep the website responsive across mobile, tablet, and desktop.
- Keep the visual design elegant, modern, and tasteful at all times.
- Preserve strong performance and fast loading.
- Use images carefully and optimize them.
- Avoid unnecessary complexity, heavy dependencies, analytics, logging, or backend services.

## Preferred Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion only when motion materially improves polish and experience

## Design Expectations
- Avoid generic, template-like layouts.
- Use intentional typography, spacing, composition, and motion.
- Favor clean hierarchy and high-end visual presentation.
- Treat responsiveness as part of the design, not an afterthought.

## Working Style
- Make direct changes unless clarification is required.
- Keep explanations concise and practical.
- Optimize for maintainability as well as aesthetics.
